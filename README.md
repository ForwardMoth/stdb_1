# Вторая часть проекта реализованная на СУБД SQLite

## Здесь поддерживаются операции:

### Для людей:

- Добавление
- Просмотр
- Редактирование информации
- Удаление
- Поиск по фамилии

### Для телефонов:

- Добавление
- Просмотр по человеку 
- Редактирование 
- Удаление

### Для групп:

- Добавление
- Просмотр
- Редактирование информации
- Удаление
